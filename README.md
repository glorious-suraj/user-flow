# UserFlow App

A React Native app built with TypeScript.

Features:
- User Registration & Login
- Form validation using Formik + Yup
- Local session storage with AsyncStorage
- Posts API integration
- Redux Toolkit state management
- Responsive Android-focused UI

Tech Stack:
- React Native CLI
- TypeScript
- Redux Toolkit
- React Navigation
- Axios
- AsyncStorage

API Used:
https://jsonplaceholder.typicode.com/posts
